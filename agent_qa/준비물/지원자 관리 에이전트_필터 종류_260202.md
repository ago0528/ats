아래는 **“필터(검색 조건) 생성/구성에 직접 쓰이는 도구”만** 추려서 리스트업한 거야.
(즉, URL 생성/검색 실행/실행 알림/분석 코드 실행/공통 데이터 검색 같은 **비(非)필터 도구는 제외**)

## 필터(도구) 리스트

### A) 필터 스키마/유형 안내

* `RESUME_CONDITION_DOCUMENT` : 검색 조건 생성용 JSON 명세(문서) 추출 (필터 구성 전 필수)

### B) 공고/채용분야/지원경로 등 “선택지 조회(필터값 소스)”

* `RESUME_FINDER_RECRUIT_PLAN_LIST` : 채용플랜 목록 조회
* `RESUME_FINDER_RECRUIT_NOTICE_LIST` : 공고 목록 조회
* `RESUME_FINDER_RECRUIT_SECTOR_LIST` : 채용분야 목록 조회 (recruitNoticeSn 필요)
* `RESUME_FINDER_APPLY_CHANNEL_LIST` : 지원 경로 목록 조회

### C) 조건 “생성” 도구 (검색조건 객체를 만들어 conditionList에 넣는 용도)

* `RESUME_FINDER_NATIONALITY_CONDITION` : 국적 조건 생성
* `RESUME_FINDER_CAREER_CRITERIA_CONDITION` : 경력 기준 조건 생성
* `RESUME_FINDER_FOREIGN_LANGUAGE_SKILL_CONDITION` : 외국어 활용 능력 조건 생성
* `RESUME_FINDER_OVERSEAS_EXPERIENCE_CONDITION` : 해외 경험 조건 생성
* `RESUME_FINDER_ACTIVITY_CONDITION` : 학내외 활동 조건 생성
* `RESUME_FINDER_VOLUNTEER_CONDITION` : 봉사활동 조건 생성
* `RESUME_FINDER_FINAL_VALUER_CONDITION` : 최종 평가자 조건 생성
* `RESUME_FINDER_SCREENING_CONDITION` : 전형 조건 생성 (채용플랜 + 전형유형 필수)
* `RESUME_FINDER_GENERAL_VALUER_CONDITION` : 일반 평가자 조건 생성

### D) 학력/자격 등 “텍스트 검색 → 값 선택(필터값 소스)”

* `RESUME_FINDER_HIGH_SCHOOL_LIST` : 고등학교 검색
* `RESUME_FINDER_COLLEGE_LIST` : 대학교 검색
* `RESUME_FINDER_COLLEGE_MAJOR_LIST` : 대학교 전공 검색
* `RESUME_FINDER_GRADUATE_SCHOOL_LIST` : 대학원 검색
* `RESUME_FINDER_GRADUATE_SCHOOL_MAJOR_LIST` : 대학원 전공 검색
* `RESUME_FINDER_LICENSE_LIST` : 자격증 검색
* `RESUME_FINDER_FOREIGN_EXAM_LIST` : 외국어 시험 검색

---

### 참고: “필터가 아닌 것(제외한 항목)”

* `RESUME_CONFIRM_URL` (화면 URL 생성)
* `RESUME_FINDER` (검색 실행/지원자 수 조회)
* `ASSISTANT_EXECUTION_PROGRESS_NOTIFY` (실행 과정 알림)
* `RESUME_ANALYSIS_CODE_RUNNER` (분석 코드 실행)
* `RESUME_COMMON_DATA_SEARCH` (공통 데이터 문맥 검색 워커 호출)
