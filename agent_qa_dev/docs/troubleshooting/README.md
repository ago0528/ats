# 백오피스 트러블슈팅 가이드

이 폴더는 백오피스 운영 이슈를 재현/분석/재발방지까지 관리합니다.

## 이슈 템플릿
- 파일명: `docs/troubleshooting/incidents/YYYYMMDD-이슈명.md`
- 필수 섹션
  - `개요`
  - `재현 경로`
    - 실행 요청/파라미터 예시
  - `재현 데이터`
  - `원인 추정`
  - `조치`
  - `재발 방지`
    - 테스트 추가 여부
    - 배포 게이트 업데이트 여부

## 운영 규칙
- 핵심 플로우(템플릿 다운로드, 질의 실행, LLM 평가, 프롬프트 조회/수정)는 버그 발생 즉시 이슈화
- 해결되지 않은 이슈가 남아있는 경우 동일 범위 기능 배포를 금지
